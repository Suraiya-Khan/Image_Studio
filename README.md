# 🖼️ Image Processing Studio  

Turn your images into art with ease! The **Image Processing Studio** is a Streamlit-powered web app that lets you upload, edit, and transform images in real time — all from your browser.  

## 🚀 Features  
- **Resize** – Adjust width and height to custom dimensions  
- **Crop** – Select and cut out specific regions interactively  
- **Flip** – Mirror images horizontally, vertically, or both  
- **Filters** – Apply grayscale, sepia, or negative effects  
- **Download** – Save your edited image instantly  

## ⚙️ Tech Stack  
- **Python**  
- **Streamlit** for UI  
- **OpenCV** for image processing  

## 🎯 Why Use It?  
This project is perfect for:  
- Students learning computer vision basics  
- Developers experimenting with OpenCV  
- Hobbyists who want a lightweight photo editor  

## 📦 Installation  
```bash
git clone https://github.com/your-username/image-processing-studio.git
cd image-processing-studio
pip install -r requirements.txt
streamlit run app.py
```
